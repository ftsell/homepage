Pixelflut is a small game for programmers inspired by reddits r/place and CCCGOE's implementation.

You can participate by sending a TCP or UDP package to finn-thorben.me:1234 in the format PX $X $Y $COLOR\n.
$X is the x coordinate (maximum = {{xSize}})
$Y is the y coordinate (maximum = {{ySize}})
$COLOR is the HEX encoded rgba value where a is optional (no '#' character)

More information is available by sending HELP\n to the same server.
